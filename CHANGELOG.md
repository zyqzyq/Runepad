# 更新日志

本文件记录 Runepad 项目的所有重要变更。

格式参考 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.1.0/)。

---

## [未发布]

### 新增
- 新增 `docs/screenshots/` 截图目录，并在 README 中预留截图展示区。

### 变更
- 优化 Markdown 文件在编辑器中的阅读间距与 token 视觉细节。

### 修复
- 区分标题栏最大化与还原状态的图标和提示文案，并补齐菜单按钮可访问性状态。

## [0.6.2] - 2026-05-29

### 新增
- 编辑器支持 JSON、Python、YAML 文件的代码折叠栏与默认折叠快捷键。
- 设置页支持选择编辑器语法高亮主题。

### 修复
- 修复已识别语言的文件未显示 CodeMirror 默认语法高亮的问题。
- 修复编辑器区域内 `Ctrl+F` 被窗口级快捷键二次处理导致查找面板无法打开的问题。
- 将替换快捷键调整为可二次关闭的 `Ctrl+R`，并修复搜索面板遮挡顶部菜单的问题。

## [0.6.1] - 2026-05-29

### 新增
- 持久化侧边栏显隐状态和窗口尺寸位置。
- 当外部修改了当前正在编辑的脏文件时，弹出确认重载对话框。
- 根目录 `CHANGELOG.md` 自动生成，记录全版本历史。
- Release Workflow 自动根据版本号读取 `CHANGELOG.md` 填充 Release Body。

## [0.6.0] - 2026-05-26

### 新增
- 发布 0.6.0 版本。
- 添加 MIT 开源许可证。
- 打包时从环境变量读取发布者（Publisher）信息。

### 修复
- 修复双击文件打开时窗口未正常显示的问题。

## [0.5.0] - 2026-05-21 ~ 2026-05-25

### 新增
- 支持通过双击文件或右键菜单 "Open with Runepad" 打开文件。
- NSIS 安装包右键菜单钩子。
- 支持 MSIX 打包与 Windows 11 右键菜单集成。
- Shell 扩展支持中文。
- 快捷键支持。
- 单窗口模式（禁止多实例）。
- 编辑器注释/取消注释快捷键（`Ctrl+/`）。
- 核心 Rust 逻辑与前端流程的自动化测试。

### 变更
- 移除 `get_system_theme` IPC 命令，统一使用 WebView `matchMedia` 检测系统主题。

### 修复
- 修复若干 UI 与打包相关的问题。

## [0.4.1] - 2026-05-21

### 新增
- 添加 GitHub Actions 自动发布工作流。

### 修复
- 修复 Tauri release action 版本号错误。

## [0.4.0] - 2026-05-19 ~ 2026-05-20

### 新增
- 设置对话框：支持主题、字体、语言和国际化（i18n）。
- 非阻塞式会话保存与可靠的窗口关闭处理。
- 极简扁平化 UI 壳层与统一编辑器主题风格。

### 变更
- 优化发布版本的启动与会话恢复性能。
- 更新应用图标与窗口标题。
- 重构 `AGENTS.md`，将详细文档拆分到 `docs/` 目录。

### 修复
- 修复会话恢复后产生空 Tab 的问题。
- 编辑器懒加载 CodeMirror，加快会话恢复速度。

## [0.3.0] - 2026-05-18

### 新增
- 侧边栏支持目录树浏览。
- 支持多编码读写（UTF-8、GBK 等）。
- 查找 / 替换面板。
- 最近文件列表。
- Tab 拖拽排序。
- 会话恢复（预览 + 完整恢复）。
- 外部文件变更监听与提示。

## [0.2.0] - 2026-05-18

### 新增
- 核心类型定义、Zustand 状态管理与编辑器实例映射。
- Rust 后端文件读写命令与路径安全校验。
- 封装 IPC 接口（文件与系统操作）。
- 主题桥接与 CodeMirror 6 主题集成。
- 应用布局壳层与侧边栏占位。
- CodeMirror 编辑器面板与元数据监听。
- Tab 栏交互与多文档生命周期管理。
- 文件新建、打开、保存流程。
- 快捷键：`Ctrl/Cmd + O/S/N/W`。
- 状态栏：编码格式、光标位置、字数统计。
- 应用菜单。

### 变更
- 升级到 Tailwind CSS v4。
- 接入 shadcn/ui v4 并配置路径别名。
- 升级 Zustand 到 v5。

### 修复
- 修复打开文件内容为空的问题。

## [0.1.0] - 2026-05-18

### 新增
- 初始化项目脚手架。
- 添加 P0 前端依赖与 lockfile。
